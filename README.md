# SmartWeave Console

A console for interacting with SmartWeave.

## Usage

```shell script
$ yarn console
```

```shell script
smartweave > 
*********************************
*       Smartweave Console      *
*********************************

Press enter to start.


smartweave > await readContract(arweave, "qcxXgQxu1XuHDMYJW9adKn9P8a06_KWEwwyr2y0vkZs")
```
